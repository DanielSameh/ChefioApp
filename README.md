  
# ChefioApp
An implementation for App UI Kit with react native
Chefio is an application helping you to find the best recipes you need and can scan any ingredients you want in it or even to
know more about other recipes that your friends have.
The design inspired by: https://www.figma.com/file/hlaIuzH51CC7lvwU4EP3Py/Chefio---Recipe-App-UI-Kit

# Tools & Technologies Used in project
- React Native to implement the application in both platform IOS and Android <br>
- Design 10+ Screens and 20+ reusable Components for the app <br>
- Axios to make requests and intergrate RESTFUL Api <br>
- Structuring folders and writing a clean code <br>

# Installation
- Download The repo <br>
- Run commender in the folder <br>
`npm install`<br>
`npm start`
